✨ 학습 도구로서의 가치

이 프로그램은 단순한 이득을 위한 도구가 아닌, **컴퓨터 비전(Computer Vision)**과 **AI 제어 로직**을 이해하기 위한 학습 자료입니다. 봇이 화면에서 대상을 **어떻게 인식**하고 **어떻게 조준**하는지 그 원리를 투명하게 분석할 수 있습니다.

프로젝트의 최종 목표는 **인공지능 기반의 게임 환경 분석 및 제어 기술**을 배우고 싶은 개발자, 학생, 연구자들에게 실질적인 코드와 동작 원리를 제공하는 것입니다.

🔎 코드 분석: AI 작동 원리 해부

제공된 파이썬 코드(`main.py`)의 주요 작동 원리를 3단계로 나누어 설명합니다. 실제 코드 줄과 기능을 연결하여 학습 효과를 극대화하세요.

1. 관심 영역(ROI) 캡처 및 전처리 (grab_screen)

화면 전체가 아닌 중앙 **700x700 픽셀** 영역만(`BOX_SIZE`) 캡처하여 **처리 속도**를 높입니다. 캡처된 이미지는 **HSV 색 공간**으로 즉시 변환되어 타겟 탐색 준비를 마칩니다.

2. 픽셀 기반 타겟 판별 (cv2.findContours)

**빨간색 픽셀** 마스크를 생성한 후, 윤곽선(`Contours`) 분석을 통해 **종횡비(W/H > 3.0)** 필터를 적용합니다. 이 필터링 과정은 적 팀의 **체력 바**를 다른 빨간색 요소(폭발 이펙트, UI)와 구분하는 핵심입니다. **cv2.boundingRect**를 통해 최종 타겟 영역을 확정합니다.

3. 속도 제어와 마우스 입력 (move_mouse)

계산된 조준 오프셋을 **`smoothness = 30`**으로 나누어 마우스 이동량을 **감속**시킵니다. 이 '휴먼 모드' 로직은 마우스 움직임의 **미분 값(Derivative)**을 제어하는 원리를 보여주며, 기계적인 움직임을 **부드러운 인간의 동작**처럼 보이게 하는 **제어 공학**적 측면을 학습할 수 있습니다.

❓ 자주 묻는 질문 (FAQ)

Q. 이 프로그램이 일반적인 핵(Hack)과 다른 점은 무엇인가요?

A. 일반 핵은 **메모리(Memory)에 직접 접근**하여 데이터를 조작하지만, 저희 프로그램은 게임 외적인 영역인 **화면 픽셀(Pixel)**만을 분석합니다. 이는 AI 학습 원리를 보여주는 데 초점을 맞춘 순수한 **외부 제어 학습 도구**입니다.

Q. AimbotController.exe는 어떤 파일인가요?

A. **`main.py`** 소스 코드를 **실행 가능한 파일(.exe)**로 변환한 것입니다. Python 환경 구축 없이 바로 실행해서 프로그램의 동작을 확인해 볼 수 있습니다. (Windows 전용)

Q. 코드를 수정하려면 어떤 툴이 필요한가요?

A. Python 설치, VS Code 또는 PyCharm 같은 **코드 에디터**, 그리고 `pip install opencv-python numpy mss pywin32` 명령어로 라이브러리만 설치하면 됩니다.
