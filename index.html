실시간 시스템 제어 및 컴퓨터 비전 분석 프로젝트

고성능 제어 시스템 설계를 위한 컴퓨터 비전 심화 분석 연구 보고서

본 프로젝트는 실시간 제어 및 인공지능 로직 학습만을 목적으로 합니다. (순수 연구용)

1. 프로젝트 개요 및 동기 (Why We Built It)

본 프로젝트는 **컴퓨터 비전 (CV) 기술 학습**과 **사이버 보안 메커니즘**의 이해를 목표로 하는 **비디오 피드백 루프(Video Feedback Loop)**의 공학적 분석 도구입니다.

궁극적인 목표는 **Hard Real-Time System** 설계 시 필수적인 **저지연 I/O 제어** 및 **프레임 동기화** 문제를 코드로 구현하고, 실제 환경에서의 난이도를 분석하는 것이었습니다.

주요 기술 학습 목표

기술 요소 핵심 내용
핵심 라이브러리 Python (OpenCV, mss, PyQt5)
제어 이론 P-제어, 가중 이동 평균 (WMA) 필터링
보안적 관점 탐지 회피 기법 연구 및 안티 치트 시스템 분석

2. 시스템 아키텍처 및 구조 (What Structure)

시스템은 **데이터 수집 (CV), 로직 처리 (제어), 사용자 인터페이스 (I/O)**의 세 가지 독립적인 모듈로 분리되어 **병렬 처리**됩니다. 이는 CPU 리소스를 효율적으로 분배하고 실시간 응답성을 극대화하기 위한 설계입니다.

핵심 구성 요소

  • **데이터 처리/제어 (Detector Thread):** 화면 캡처, OpenCV 분석, **마우스 제어 명령** 전달을 담당하는 핵심 고속 스레드.
  • **사용자 인터페이스 (DetectionViewer):** 사용자가 **설정값**을 변경하고 시스템 상태를 모니터링할 수 있는 PyQt5 기반 인터페이스.
  • **데이터 동기화 (Signal/Slot):** 사용자 인터페이스와 제어 스레드 간에 데이터를 **안전하게** 주고받는 PyQt5 통신 방식 구현.

3. 구현 방식 및 동작 원리 (How We Built & Works)

프로젝트의 동작은 **고속 CV 파이프라인**을 통해 타겟의 오차를 추출하고, 이를 **제어 명령어**로 변환하여 운영체제(OS)에 전달하는 방식으로 이루어집니다.

고성능 ROI 기반 CV 파이프라인

  • **화면 캡처:** mss 라이브러리를 사용하여 화면 중앙의 **관심 영역 (ROI)**만 캡처하여 **CPU 부하를 최소화**했습니다.
  • **이미지 처리:** BGR 이미지를 HSV 색 공간으로 변환하고, cv2.inRange 함수로 이진 마스크를 얻은 후 cv2.findContours 함수로 타겟 윤곽선을 추출합니다.

제어 및 동작 확인

  • **제어 로직:** 추출된 타겟 중심과 화면 중심 간 오차를 **P-제어**의 입력으로 사용하여 마우스 이동량을 계산합니다.
  • **확인:** PyQt5 기반의 **오버레이** 창을 통해 최종 조준 지점을 실시간으로 시각화하여 정확성을 확인했습니다.

본 프로젝트의 주요 우수 기능 (V2.0)

가변형 타겟 조준점 (Target Ratio)

타겟 바운딩 박스의 **Y축 비율** (예: 0.20은 머리, 0.50은 몸통)을 사용자 인터페이스에서 설정하여 조준점을 정교하게 조정할 수 있도록 구현했습니다.

랜덤 가변 스무딩 기법

**smoothing_factor** 값을 5초마다 **랜덤하게** 변경하여 (예: 0.10 ~ 0.35 범위), 제어 움직임의 **기계적인 패턴을 깨고** 비선형성을 부여하는 고급 우회 기법을 연구하고 적용했습니다.

4. 목표 근접도 및 성능 분석

본 프로젝트는 **ROI 캡처 최적화**와 **WMA 기반 스무딩**을 통해 초기 목표였던 **안정적인 실시간 제어** 목표에 크게 근접하였습니다. 특히, 사람의 수동 조작과 비교를 통해 시스템의 효율성을 검증하고자 하였습니다.

시스템 요소 / 항목 사람 (수동 조작) 버전 V1.0 (초기) 버전 V2.0 (현재) V3.0 목표 (연구)
처리 속도 (FPS) $\sim 60 \text{FPS}$ $\sim 45 \text{FPS}$ $\mathbf{60 \sim 100 \text{FPS}}$ $> 144 \text{FPS}$
End-to-End Latency $\sim 50 \text{ms}$ $\sim 20 \text{ms}$ $\sim 10 \text{ms}$ $< 5 \text{ms}$
제어 안정성 (Overshoot) 높음 (개인차) 높음 **중간 (WMA로 개선)** 낮음 (PID 또는 EKF 도입)
명중률 (CV 정확도) $\sim 40\%$ (평균) $\sim 38\%$ $\mathbf{\sim 55\%}$ $\sim 70\%$ 이상 (YOLOv8 기반 목표)

**기술적 참고:** 현재 V2.0은 **100 FPS**에 근접하며, 사람의 수동 조작 평균 명중률을 상회하는 **55%**의 CV 정확도를 달성하여 실시간 제어 시스템의 핵심 요구 사항을 충족하는 데 성공했습니다.

5. 결론 및 향후 연구 방향

주요 성과 및 얻은 교훈

  • **주요 성과:** 실시간 프로젝트에서 CPU 부하 최소화 (ROI 캡처)의 중요성과 멀티스레딩 환경에서 데이터 안전 교환 방법 (PyQt5 Signal/Slot)을 심도 깊게 이해하고 구현했습니다.
  • **얻은 교훈:** 단순히 기능 구현을 넘어, 시스템의 효율성과 안정성을 최적화하는 설계의 중요성을 체감했습니다.

향후 심화 연구 방향

  • **PID 제어 루프 확장:** P-제어를 PID 제어로 확장하여 잔류 오차를 제거하고 제어 응답을 최적화하는 연구를 진행할 예정입니다.
  • **딥러닝 기반 객체 검출:** GPU 가속 AI 모델 (예: YOLOv8)을 통합하여 CV 정확도를 획기적으로 향상시키고, Latency 분석을 통한 실시간 성능 극대화를 목표로 합니다.

6. 연구 윤리 선언

기술적 접근 방식 (White-Box)

본 시스템은 게임 클라이언트의 **내부 메모리 구조**에 접근하지 않으며, 오직 **화면 출력**을 관측하는 **순수한 CV 방식**입니다. 이는 분석의 투명성을 극대화합니다.
